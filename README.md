# ACC102 Artificial Intelligence-Driven Data Analytics
## Financial Analysis of Midea, Gree, Haier (2021–2024)

### 1. Project Purpose
This project uses Python to conduct financial data analysis and visualization for three leading Chinese home appliance listed companies: Midea, Gree, and Haier. It focuses on revenue trends, gross profit margins, and net profit margins from 2021 to 2024.

### 2. Data Source
- Midea Group (000333)
- Gree Electric Appliances (000651)
- Haier Smart Home (600690)
- Period: 2021–2024
- Raw data: operating revenue, cost of goods sold, net profit from official annual reports
- All ratios are calculated using standard accounting formulas

### 3. Tools & Libraries
- Python
- pandas: data processing
- matplotlib: data visualization
- numpy: numerical calculation

### 4. Analysis Content
- Revenue trend comparison (2021–2024)
- 2024 revenue, gross margin, net margin comparison
- Average revenue market share
- Overall profitability comparison

### 5. Key Findings
- Midea has the largest revenue scale and continuous growth.
- Gree has the strongest profitability performance.
- Haier maintains stable revenue growth with steady margins.
